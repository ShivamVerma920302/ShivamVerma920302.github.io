Movie Showtime Finder


[ASU, Fall 2018]


Summary

MovieShowTime Finder is an intelligent web application which analyzes the user data, maps it to the new releases according to the user interest and recommend movies by sending mail or text notifications. Movie Showtime Finder uses a content based recommendation engine which suggests movies based the similarity measure of the user selected movies.

App Features

  • 3rd Party Autthentication from Twitter, Google and Facebook using OAuth.
  • User Login/Signup implemented using Django Authentication system.
  • User can search for movie playing in threatre nearby.
  • Customized Home page for a user which diplays movies according to their interests.
  • User can manage movies displayed on their home by selecting the genre and movies they like.
  • A Python script, running periodically using AmazonEC2 instance to check the status of the new movies released.
  • An email is being sent to all the users if the newly released movie matches the interest of the user .
  • Designed an efficient and scalable database using Azure SQL Database.
  • Used BeautifulSoup for efficient web page scrapping to display the information on movies on user homepage.
  • Wrote python script to calculate the cosine similarity matrix based on features such as cast, crew, genre, movie description, etc. The matrix is updated every time new movie is added to the theatres and recommendations are produced at run time.
  • Used AzureKeyVault to securely access the secrets and passwords for database and mailing utlities.
  • Deployed the web application on Azure using Docker Image.

Tech Used:

Utilized: Python, Django, AzureSQL, AzureKeyVault, Numpy, Pandas, Scikit-Learn, OAuth Authentication, Docker, AzureWebApp, JavaScript, HTML, CSS, Bootstrap

Sentiment Analysis of Text


[Dec 2018]


Summary

Wrote scripts to computationally identify and categorize opinions expressed in a piece of text. Compared the accuracy of the classfier on using unigram, bigram and trigram.

App Features

  • Used a combination of Natural Language Processing and Text Analytics on the feedback to computationally identify and categorize opinions expressed in a piece of text.
  • Cleaned and prepared data by removing numbers, URLs, links, punctuations and the meaningful text was extracted using Suffix dropping algorithm, Lemmatization algorithms and n-gram analysis.
  • Classified the text into negative and positive based Logistic Regression.
  • Compared and Plotted the accuracy of the classfier on using unigram, bigram and trigram.

Tech Used:

Python, NLTK, Numpy, Pandas, Scikit-learn, Jupyter-Notebok

Vulnerablity Analysis


[ASU, Fall 2018]


Summary

Our aim is to develop an linked application that analyses the semantic data associated with malware and vulnerabilities to generate actionable insights. These insights will provide the users with information as to the characteristics of the attacks and exploits commonly being carried out today; the types of attacks being carried out and the technologies being targeted by attackers while carrying out these attacks. The aim of this application is to use linked data technologies to perform statistical analysis on the enriched and readily available semantic data about vulnerabilities and develop a type of decision support system, thus providing the users with insights and key indicators, helping them to make decisions with regards to the efforts needed for vulnerability identification, mitigation and prevention.

App Features

  • Developed a linked application that analyses the semantic data associated with malware and vulnerabilities to generate actionable insights.
  • Used CVE Descriptions, CVSS Parameters, CWE and CPE in conjunction with associated data from DBPedia pertaining to cyber security domain thus helping userto make decisions with regards to the efforts needed for vulnerability identification, mitigation and prevention.

Tech Used:

Protege, Tableau, Python, BeautifulSoup, Pandas, HTML/CSS, Bootstrap
